HTML images

The html img is used to embed an image in a web page, this tag is also empty, contains attributes only, and doesn't contain a closing tag.

The img tag has two required atrributes

The <image> tag is empty, it contains attributes only, and does not have a closing tag.

  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image

    • The syntax woul be like <img src="url" alt="alternatetext">

    The src attribute:

    The src attribute specifies the path (URL) to the image. For example:

    <img src="img_chania.jpg" alt="Flowers in Chania">



    The alt Attribute:

    The alt attribute provides an alternate text for an image, if the user for some reason cannot view it. The value of the alt should describe the text, for example:

    <img src="img_chania.jpg" alt="Flowers in Chania" width="460" height="345">



    Width and Height, or Style:

    The width, height, and style attributes are all valid in HTML.

    However, using the style attribute is better because prevents style sheets from changing the size of the images

    Common Image Formats:

    The most common image file types, which are supported in all browsers are:

    Abbreviation File Format Extension
    APNG Animated Portable Network Graphics .apng
    GIF Graphics Interchange Format .gif
    ICO Microsoft Icon .ico, .cur
    JPEG Joint Photographic Expert Group image .jpg, .jpeg, .jfif, .pjpeg, .pjp
    PNG Portable Network Graphics .png
    SVG Scalable Vector Graphics .svg

    HTML Image Tags

    Tag Description
    <img> Defines an image
    <map> Defines an image map
    <area> Defines a clickable area inside an image map
    <picture> Defines a container for multiple image resources

    HTML Favicon:

    How To Add a Favicon in HTML:

    You can use any image you like as your favicon

    You can also create your own favicon on sites like https://www.favicon.cc

    A favicon image is displayed to the left of the page title in the browser tab

    To add a favicon to your website, either save your favicon image to the root directory of your webserver, or create a folder in the root directory called images, and save your favicon image in this folder.

    Next, add a <link> element to your "index.html" file, after the <title> element.

    Favicon File Format Support

    This table shows the file format support for a favicon image:

    Safari Yes Yes Yes Yes Yes
